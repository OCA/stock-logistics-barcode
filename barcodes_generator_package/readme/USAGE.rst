To use this module, you need to:

* Go to a Packaging form:

1 for manual generation
    * Set a Barcode Rule
    * Set a Barcode Base
    * click on the button 'Generate Barcode'

2 for automatic generation
    * Set a Barcode Rule
    * click on the button 'Generate Base'
    * click on the button 'Generate Barcode'
